# 日期切換與飯店設定 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 讓旅行前測試與旅途中查看都能手動切換日期，並把飯店地址集中設定後套用到 Google Maps 路線。

**Architecture:** 延續現有 vanilla JavaScript 單檔架構，不拆新模組。新增本機 `selectedDayIndex` 狀態與小型設定物件，所有今日角色、今日行程、候補加入/移除、角色頁與候補池動作共用目前選取日期。

**Tech Stack:** HTML, CSS, vanilla JavaScript, browser localStorage.
